Running the Job

Overview

The construction of your Job is complete, and now you are ready to run it and see the results.

Examine input file

Before running your Job, examine the input file again..

  1. LOCATE THE FILE

    Use the File Browser to locate the Custs.csv file in the C:/StudentFiles folder :

  2. OPEN IT IN A TEXT EDITOR

    Right-click the file and select Edit with Notepad++:

    You can notice that several states are in lower case. Your Job is designed to change all states in upper case.
    Now, you can run your Job and check the output.

Run

  1. OPEN THE RUN VIEW

    Click the Run view under the design space:

    This is where you execute a Job. Note that in the design workspace, the name of the Job in the tab at the top is preceded by an asterisk, indicating unsaved changes.

  2. RUN THE JOB

    Click the Run button:

    Note that statistics about the Job execution display in the design workspace:

    3. This shows that 100 rows passed through each of the connections.

  3. CHECK THE OUTPUT FILE

    In the File Browser, locate and then open the output file CappedOut.csv in C:/StudentFiles. Notice that all state abbreviations are now in uppercase and then close the file:
