Combining Columns

Overview

The tMap component provides the capability to perform a variety of data mapping and transformation functions. Imagine that your destination data store requires the first and last names to be combined in a single column. Follow the steps below to achieve that.

Edit Schema

  1. OPEN THE tMap COMPONENT CONFIGURATION WINDOW

    Double-click the tMap component.

  2. SELECT row1.Last IN THE OUTPUT TABLE

    Click the row1.Last expression in the CappedOut table:

    In the Schema editor at the bottom of the window, notice that the same column is selected:

    Because you are going to combine the data from two columns into a single column, this column is no longer necessary.

  3. DELETE THE UNNEEDED COLUMN

    Click the Remove selected items button, marked with a red X (highlighted in the previous print screen ), to delete the Last column from the output schema:

    Important ! Make sure to click Remove selected items in the Schema Editor of CappedOut (lower right part of the Map Editor) and not the Remove table red X (in the upper right part of the Map Editor). If you click Remove Table, you will delete your mapping and will have to rebuild it from the beginning.

  4. The column has disappeared from the CappedOut table at the top of the window as well.

Combine Columns

  1. MAP BOTH FIRST AND LAST NAMES TO THE SAME OUTPUT COLUMN

    Drag the column Last from the row1 table on the left, dropping it onto First in the CappedOut table:

    Note that the expression now contains references to two columns from the row1 table.

  2. CONCATENATE FIRST AND LAST NAMES

    Insert a plus sign (+) between the two references:

    Because the two columns contain strings, the plus sign concatenates the values.

  3. ADD A SPACE BETWEEN THE FIRST AND LAST NAMES

    To include a space between the first and last names, insert " " + after the first plus sign. Press the Enter key on your keyboard to save this expression :

  4. CHANGE THE COLUMN NAME AND UPDATE THE STRING LENGTH

    In the Schema editor at the bottom of the window, click First for the CappedOut table and then change it to Name:

    Then update the length of the Name field from 15 to 30.

    Now the schema column name in the CappedOut table is Name rather than First:

  5. SAVE YOUR CHANGES

    Click Ok to save your changes, and then propagate the changes when prompted.

Run

  1. RUN THE JOB

    In order to Run the Job you can either use the Run button or press the F6 key on your keyboard.

  2. OPEN THE DATA VIEWER

    Right-click the tFileOutputDelimited component and select Data viewer:

  3. CHECK THE OUTPUT FILE

    Explore the content of the output file using the Result Data Preview display:


    Notice that the first column is now Name and is the combination of First and Last columns of the Custs.csv file separated by a space (as specified in the Expression Builder).

  4. YOU CAN ALSO VIEW THE DATA IN RAW CSV

    Click on the File Content tab to see the raw format of the file:

    Click Close.

Next

You have now finished this Job. Next you will Duplicate the Job , modify it and then execute the new Job.
