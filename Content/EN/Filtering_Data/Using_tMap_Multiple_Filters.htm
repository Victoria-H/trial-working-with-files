Using tMap for Multiple Filters

Overview

You will now learn how to filter on several states in one tMap component, following best practices. You will create 3 output files each containing a different state.

Duplicate Job

  1. DUPLICATE THE Filter_States JOB

    Right-click on the Job Filter_States and click Duplicate:

  2. GIVE THE JOB A NAME

    Name the new Job Filter_States_tMap and click OK:

  3. OPEN THE JOB

    Double-click the newly created job to open it.

Edit Output

  1. COPY THE EXISTING OUTPUT COMPONENT

    Enter a new output component to the Job. Right-click on tFileOutputDelimited and click Copy:

  2. PASTE IT INTO THE DESIGN WORKSPACE

    Right-click below the output component in the canvas and click Paste to place the new tFileOutputDelimited component:

  3. PASTE AGAIN TO CREATE 3 OUTPUT COMPONENTS

    The component is placed below the existing tFileOutputDelimited component.
    Below the second tFileOutputDelimited component right-click and select Paste again to drop the third tFileOutputDelimited component to your Job:

  4. You have placed three outputs now in your Job:

  5. CHANGE THE OUTPUT FILE PATH FOR THE PASTED COMPONENT

    Remember the last setting in tMap was a filter, filtering entries for CA (California) and writing the output to a file. Now you will edit the settings for the first output component and change the output file name. Double-click tFileOutputDelimited_1 and change the File Name to "C:/StudentFiles/CA_Out.csv":

  6. LABEL THE COMPONENT

    Click View to change the output name to CA. Enter CA into the field Label format:

  7. CHANGE THE OUTPUT FILE PATH FOR THE SECOND PASTED COMPONENT

    Now double-click the second tFileOutputDelimited component, change the file name to NY_Out.csv and enter NY for the Label format:

  8. CHANGE THE LABEL

    Enter the changes for the third tFileOutputDelimited component to AK:

  9. ADD A NEW OUTPUT ROW FROM THE tMap COMPONENT

    Right-click the tMap component and click Row then *New Output* (Main):

  10. CONNECT THE TO THE NY OUTPUT COMPONENT AND NAME THE OUTPUT ROW

    Connect it to the NY component, name it NY and then click OK:

  11. GET THE SCHEMA OF THE NY OUTPUT COMPONENT

    Click Yes when prompted Do you want to get the schema of the target component?:F

  12. CONNECT tMap TO THE AK OUTPUT COMPONENT IN THE SAME WAY

    Repeat these steps for the last output component AK:

Edit tMap

  1. OPEN THE tMap MAPPING EDITOR

    Double-click the tMap component to open the Mapping Editor.

  2. USE Auto map! TO MAP THE INPUT TO THE NEW OUTPUT

    Click the Auto map! button on the top right:

  3. The system will map the elements as follows:

    The expression combining first and last names is missing in the new output as is the UPCASE expression.

  4. COPY THE NAME EXPRESSION FROM THE CappedOut TABLE

    You can copy and paste Expressions. Right-click on the expression for the field Name in the CappedOut table and click Copy:

  5. PASTE THE EXPRESSION IN THE NY TABLE

    Right-click the expression for the field Name in the output NY and click Paste:

  6. PASTE IN THE AK OUTPUT TABLE

    Paste it also in the Name column for the output AK:

  7. REPEAT THIS FOR THE STATE EXPRESSION

    Copy and paste the expression for the field State also:

  8. Your output tables should look as follows:

  9. COPY THE FILTER EXPRESSION FROM THE FIRST OUTPUT TABLE

    If it is not already opened, click the Activate / unactivate the expression filter Icon on the top right of the output schema CappedOut and copy the expression "CA".equals(row2.StateCode):

  10. PASTE THE EXPRESSIO IN THE NY TABLE AND MODIFY IT

    Click the Activate / unactivate the expression filter Icon on the top right of the output schema NY and paste the expression to it. Then change it to "NY".equals(row2.StateCode) :

  11. REPEAT THIS FOR THE AK: OUTPUT

    Apply the same changes for the output AK:

  12. SAVE YOUR CHANGES AND SAVE THE JOB

    Click OK to save your tMap editor changes, and then save the Job.

Run Job

  1. RUN THE JOB

    Run the Job and notice how the results are written to different output files based on the filters put in place:

  2. CHECK THE CSV OUTPUT FILES

    Navigate to and display the output files in C:/StudentFiles/ to verify the output. The NY_Out.csv file is displayed here:
