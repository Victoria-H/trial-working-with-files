Overview

In this lab your input will be the StateCodesMetadata and you will write to a MySQLdatabase, creating a new table.

Create New Job

  1. To create a New Job, right-click on Job Designsand then Create Standard Job:

  2. Name the new Job States2MySQL,enter your text for the Purposeand Descriptionand then click Finish:

  3. Expand the Metadata and drag the StateCodesto the Workspace. Select tFileInputDelimitedout of the list and then click OK:

  4. Expand the Db ConnectionsMetadata and drag LocalMySQL to the workspace and select tMysqlOutputfrom the list and then click OK:

  5. Right-click on the StateCodescomponent and select Rowthen Mainto connect it to the LocalMySQLcomponent:

  6. Double-click the component tMysqlOutput. The Componenttab displays the settings for it. Fill in the field Tablewith "StateMySQL". For the field Action on table,select the entry Create table if does not existfrom the dropdown list:

  7. Run the Job:

  8. Check the result in the database. Start the MySQL Workbench from the Desktop or the Menu:

  9. To open the database click Local instance MySQL56:

  10. Expand the trainingdatabase, the Tables folder and then the statemysql table:

  11. To check that your data has been written to the database trainingin the statesmysqltable, enter select * from training.statemysqlin the query and click the Icon to execute the query:

Note: The data has been written to MySQL and you can close the MySQL Workbench
