Creating a New Project

Overview

You need a new project to organize the Talend Jobs that will execute the database interactions.

Create Project

  1. Click the File menu and then click Switch Project.

  2. Click the Create... button.

  3. Name the new project "Database Training" and then provide a description.

  4. Click Finish. Your new project now appears in the project list:

  5. Select Database_Training and then click Open.

  6. When the initialization is complete, Talend Studio opens with the Welcome page displayed:

     

  7. Click Start now. The Talend Open Studio main window appears, ready for you to create Jobs:
