Getting Started with Talend Studio

Overview

This lesson will help you get started using Talend Studio. The steps guide you through the process of creating, running, and modifying a simple Talend Job that illustrates the basic features and functionality of the product. The example you build takes rows of customer data from a file, capitalizes the text in a column containing US state abbreviations, and writes the modified rows to another file:

While this is a somewhat trivial task, it demonstrates the basic sequence of extract-transform-load (ETL), where you take data from a source, perform some transformation on that data, and then send the data to a destination (which may or may not be the source itself). You will find yourself following this pattern in most of your Talend Data Integration tasks, ensuring that data from multiple sources conforms to a defined standard before being sent to a single destination for storage or processing.   

Objectives

After completing this lesson, you will be able to:

  • Start Talend Studio

  • Link Talend Studio to your Talend account, registering a new account if necessary

  • Create a Talend project to contain tasks

  • Create a Talend Job to perform a specific task

  • Add and configure components to handle data input, data transformation, and data output

  • Run a Talend Job and examine the results

Before You Begin

Be sure that you are working in an environment that contains the following:

  • A properly installed copy of Talend Open Studio for Data Integration

  • The supporting files for this lesson
