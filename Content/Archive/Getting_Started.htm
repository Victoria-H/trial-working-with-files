Starting Talend Studio

Overview

The first step in creating a Talend project is, of course, to start the software. Follow these steps to start Talend Open Studio for Data Integration.

Run

  1. Locate the file TOS_DI-win-x86_64.exe in the directory C:\TOS_DI_5.5\TOS_DI-Win32-r118616-V5.5.1:

    Note that there are different executable files for different combinations of operating system and CPU. When you are running Talend Open Studio at work or home, choose the appropriate file for your environment.

  2. An alternative is to double-click the following icon on your desktop:

  3. Double-click the file to start Talend Open Studio. The License window appears:

  4. Click Accept to agree to the license terms and continue:

  5. From this window, you choose what to open with Talend Studio: the demo project or a new project.

Create a Project

  1. Click Create to create a new project. The New project window appears:

    A Talend project is an organizational container for related elements that you want to store together. These elements serve different functions, but typically work together and can share resources.  

  2. Enter "Basic Training" into the Project Name box and a description into the Project description box:

    Only the project name is required, but it's a good idea to get in the habit of describing a project in sufficient detail that others can determine its purpose.

  3. Click Finish. You are returned to the previous window with your new project selected:

TalendForge

  1. Click Open. The Connect to TalendForge window appears:

    This window allows you to create a Talend account, if you don't already have one, and to associate this Talend Studio with your account. Your Talend account allows you to post questions or comments on Talend Forum, download shared components from Talend Exchange, visit MyTalend.com, along with a variety of other benefits.

  2. If you already have a Talend account, click existing account and then log in with that account. If not, complete the form and then click Create Account. Talend Studio opens with the Welcome page displayed:

    From this window you can read the latest company news and access other Talend locations from within Talend Studio.

Note that a browser tab also opens to the Resources page of the Talend Web site. This page provides access to many online resources you can use to learn more about Talend products. For now, just close the tab.

Start

Click Start now. The Talend Open Studio main window appears:

Register

As a Talend On-Demand Training customer, you are entitled to priority treatment when you post questions on the TalendForge forum. To register yourself as a Talend Training Customer, open this link and then follow the instructions.

Next

Now that you have a project to contain your work, you are ready to create a Job to perform the task at hand.
