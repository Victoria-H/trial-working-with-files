Utilisation des bases de données

Vue d'ensemble

Au cours de cette leçon, vous allez explorer les fonctionnalités de Talend Data Integration vous permettant d'interagir avec des bases de données.

Rappelez-vous que dans le scénario, vous prenez des données brutes de ventes provenant de différents magasins, vous effectuez une jointure sur ces données et les informations clients et produits existantes, puis vous chargez les données combinées dans une table de données des ventes combinées.

Pour une partie de la configuration de ce scénario, vous allez d'abord créer une table de base de données pour les données clients, charger ces données depuis un fichier texte dans une base de données puis répéter ce processus pour les données des produits :

Vous allez ensuite prendre les fichiers de données représentant les informations des ventes de trois magasins différents puis charger ces informations dans une table de base de données.

Finalement, vous allez effectuer une jointure entre la table des ventes et les tables clients et produits afin de créer une table maître des ventes.

Pour continuer à explorer, vous allez générer le nombre total de ventes par client et par produit.

Objectifs

Après avoir terminé cette leçon, vous pourrez :

  • Vous connecter à une base de données à partir d'un Job Talend

  • Utiliser un composant pour créer une table de base de données

  • Lire et écrire depuis et dans une table de base de données à partir d'un Job Talend

  • Filtrer les lignes de données uniques

  • Effectuer des calculs d'agrégation sur les lignes

  • Écrire des données dans un fichier XML à partir d'un Job Talend
