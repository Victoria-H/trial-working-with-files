Solution

Vue d'ensemble

Cette section présente une solution possible à cet exercice. Notez que vos solutions peuvent être différentes mais tout aussi valides.

Nouveaux contexte et variable

  1. Cliquez-droit sur LocationContextVariables dans le Repository et sélectionnezEdit context group.

  2. Cliquez sur Next.

  3. Cliquez sur Add.

  4. Modifiez le nom de la variable en RepoInputDir.

  5. Cliquez sur l'onglet Values as Table.

  6. Ajoutez les valeurs pour les deux contextes. Assurez-vous que la valeur par défaut est "C:/StudentFiles/".

  7. Cliquez surFinish.

  8. Dans la vue Contexts du Job, cliquez sur le bouton Contexts et ajoutez la nouvelle variable (cliquez sur Select All, puis sur OK).

  9. Dans la vue Component du CustomersFile, modifiez la valeur du champ File Name :

    context. RepoInputDir+ "Custs.csv"

  10. Exécutez le Job à nouveau et vérifiez que le résultat est le même que précédemment.

Étape suivante

Vous avez terminé l'exercice. Vous pouvez passer à la Conclusion.
