Création de votre premier Job

Vue d'ensemble

Dans cette section, vous allez créer votre premier Job et apprendre à ajouter et connecter des composants. Vous allez utiliser la Palette pour rechercher des composants. Vous allez également configurer les composants, dans la vue correspondante, puis exécuter le Job dans la vue Run.

 

Ce Job est très simple : il va définir un message "Hello World" qui sera affiché dans la console d'exécution.

Création du Job

  1. Pour créer un nouveau Job, dans la vue Repository, cliquez-droit sur Job Designs et sélectionnez Create Standard Job :

  2. Saisissez HelloWorld dans le champ Name et saisissez les valeurs suivantes dans les champs Purpose et Description de votre nouveau Job :

  3. Cliquez sur Finish lorsque vous avez terminé.

    Remarquez les modifications dans l'espace de modélisation graphique, la Palette et les vues de configuration.

    Le nouveau Job créé s'ouvre dans l'espace de modélisation graphique et les composants sont disponibles dans la Palette :

    Le contenu de la Palette est organisé en dossiers et est riche de plus de 800 composants, selon votre installation.

Ajout de composants

  1. Recherchez le composant tFixedFlowInput.

    Saisissez tFixed dans le champ de recherche Find Component de la Palette puis cliquez sur le bouton Search ou appuyez sur la touche Entrée de votre clavier :

    Le composant a été trouvé dans le dossier Misc.

  2. Placez votre curseur sur le composant dans la Palette pour voir une description de ce composant. Le tFixedFlowInput vous permet de définir un flux fixe pour une variable interne.
  3. Sélectionnez le tFixedFlowInput et, une fois le composant sélectionné dans la Palette, placez votre curseur dans l'espace de modélisation graphique et cliquez. Cela ajoute le composant à votre Job :

    4. Il est également possible de glisser-déposer le composant depuis la Palette dans l'espace de modélisation graphique.

  4. Pour ajouter un autre composant à votre Job, cliquez dans l'espace de modélisation graphique, ailleurs que sur le composant et commencez à saisir tLog. Une liste de composants s'affiche.

  5. Double-cliquez sur le composant tLogRow.

    6. Le composant est ajouté au Job. C'est une autre manière d'ajouter des composants à votre Job.

    La bonne pratique est de disposer vos composants dans l'espace de modélisation graphique pour imiter le flux de données et son exécution, de droite à gauche et de haut en bas.

    Notez les avertissements dans le coin supérieur droit des composants.

  6. Placez votre curseur sur l'icône rouge à côté du tFixedFlowInput pour faire apparaître une info-bulle présentant les avertissements et erreurs :

    Le composant doit être configuré et connecté à un composant de sortie.

  7. Placez votre curseur sur l'icône jaune à côté du tLogRow pour faire apparaître une info-bulle présentant les avertissements  :

  8. Vous pouvez connecter les composants.

 

Connexion des composants

 

  1. Pour relier les composants, cliquez-droit sur le tFixedFlowInput.

    Sélectionnez Row > Main et déplacez votre curseur sur le tLogRow puis cliquez sur ce composant :

  2. Le lien est créé et affiché :

    Un lien (ou connexion) permet aux données de passer d'un composant à un autre. La flèche indique la direction du flux.

    Au cours des étapes suivantes, vous allez voir comment créer un lien Main différemment.

  3. Cliquez-droit sur le lien et sélectionnez Delete :

    4. Note : il est également possible d'utiliser la touche Suppr de votre clavier pour supprimer un lien ou un composant.

  4. Pour recréer le lien, cliquez-droit sur le premier composant, maintenez le clic-droit et déplacez votre souris sur le second composant. Cela crée également un lien Main.

    5.  

    Cette méthode nécessite moins d'opérations que la précédente mais ne fonctionne qu'avec certaines connexions de type Row, selon la nature et le rôle des composants que vous connectez. Plus de connecteurs sont disponibles dans le menu contextuel, comme les déclencheurs, que vous découvrirez plus tard au cours de cette formation.

 

Configuration des composants

  1. Double-cliquez sur le tFixedFlowInput pour ouvrir sa vue Component, dans la partie inférieure de la fenêtre principale.

    Cliquez sur le bouton [...] à côté du champ Edit schema.

  2. L'éditeur Schema of tFixedFlowInput s'ouvre.

  3. Cliquez sur le bouton [+] pour ajouter une colonne.

    Modifiez son nom en Message et cliquez sur OK :

    Cliquez sur Yes lorsqu'il vous est proposé de propager les modifications.

    Cette boîte de dialogue s'ouvre à chaque fois qu'un schéma est mis à jour. Si plus d'un Job utilise ce schéma, il sera propagé à tous les Jobs.

     

  4. La colonne Message est ajoutée au schéma.

  5. Configurez la valeur (dans la colonne Value) de la ligne Message à "Hello World" :

  6. Cliquez sur l'onglet View, à gauche, dans la vue Component et, dans le champ Label format, saisissez InputMessage. Cela met à jour le nom du composant dans l'espace de modélisation graphique.

    Notez que le tFixedFlowInput n'affiche plus d'erreur, comme le schéma d'entrée a été défini et le que le composant a un lien de sortie.

  7. Double-cliquez sur le tLogRow et vérifiez les paramètres dans sa vue Component :

    Laissez les paramètres par défaut pour une première sortie dans la fenêtre de log.

  8. Notez qu'il y a un * devant le nom de votre Job. Cela signifie que le Job n'a pas encore été sauvegardé. Cliquez sur le bouton Save et le symbole * disparaît :

    Vous pouvez également appuyer sur les touches Ctrl+S pour sauvegarder votre Job.

  9. Votre Job est prêt.

Aide

  1. Cliquez sur le composant tFixedFlowInput (renommé InputMessage) et appuyez sur la touche F1 de votre clavier. Cela ouvre l'aide du composant sélectionné.

  2. Cliquez sur le lien vers la documentation du composant (cadre rouge). Double-cliquez sur l'onglet Help pour l'agrandir.

    Prenez un moment pour explorer les caractéristiques du composant. Vous pouvez constater que vous pouvez utiliser ce composant dans différents types de Jobs et comprendre ses propriétés respectives. Par exemple, si vous cliquez sur les propriétés du tFixedFlowInput, vous pouvez voir les informations concernant sa famille de composant et la manière dont configurer son schéma :

     

  3. Cliquez sur le bouton Search et trouvez la documentation du composant tLogRow.

  4. Double-cliquez sur l'onglet Help pour réinitialiser la taille de la page.

    Vous pouvez également glisser-déposer l'onglet dans n'importe quelle partie de la fenêtre principale.

    Utilisez la même méthode pour déplacer les autres onglets et organiser votre espace de travail selon vos préférences.

    Souvenez-vous que, si vous souhaitez revenir à l'était initial de votre espace de travail, vous pouvez cliquer sur Window > Reset Perspective... dans le menu principal.

Étape suivante

Vous avez créé votre premier Job dans Talend Data Integration. Vous pouvez l'exécuter.
