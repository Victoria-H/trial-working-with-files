Exécution du Job

  1. Pour exécuter le Job, ouvrez la vue Run (à côté de l'onglet Component) :

    C'est dans cette vue que vous exécutez les Jobs.

  2. Cliquez sur le bouton Run et attendez que le Job s'exécute :

  3. Le Studio Talend sauvegarde le Job, le construit et l'exécute.

    Le Job se termine rapidement, affichant les messages dans la vueRun :

    Vous pouvez voir le message Hello World affiché dans la console d'exécution.

  4. Cliquez sur le bouton Clear pour vider la console.

  5. De manière alternative, pour exécuter votre Job, vous pouvez également cliquer sur l'icône Run au-dessus de l'espace de modélisation graphique :

    Étape suivante

    Vous avez construit et exécuté votre premier Job Talend d'intégration de données.

    Vous pouvez passer à la Conclusion.
