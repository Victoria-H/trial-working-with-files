Création de métadonnées

Vue d'ensemble

Vous avez précédemment configuré un composant d'entrée de type fichier avec des informations concernant le fichier source et son schéma. Ces informations étaient locales, dans ce composant particulier. Avec le Studio Talend, vous pouvez stocker des informations de configuration, comme des métadonnées (Metadata), afin de les réutiliser dans différents composants, dans le même Job ou dans des Jobs différents. Vous allez créer des informations concernant un fichier délimité contenant des codes d'États des États-Unis et leurs noms, afin que vous puissiez utiliser les mêmes informations dans différents composants dans différents Jobs.

Création de la métadonnée

  1. Dans le Repository, développez Metadata, cliquez-droit sur File delimited et sélectionnez Create file delimited :

  2. Saisissez StateCodes dans le champ Name et les informations appropriées dans les champs Purpose et Description :

    Note : il est tentant de renseigner rapidement uniquement le nom requis, mais le respect des bonnes pratiques nécessite de renseigner les autres champs afin de documenter votre Job et vos éléments de projet, afin que les intervenants comprennent les différents objectifs.

    Cliquez sur Next.

  3. Dans la deuxième étape, cliquez sur Browse, sélectionnez *.txt dans liste des types de fichiers, sélectionnez le fichier States.txt, dans C:/StudentFiles et cliquez surOpen.

    Dans la liste Format, sélectionnez WINDOWS puis cliquez sur Next :

  4. Dans cette étape, vous renseignez les informations de configuration relatives au fichier. Sélectionnez Comma dans la liste Field separator afin de spécifier que les champs sont délimités par une virgule. Laissez les autres paramètres tels qu'ils sont, car le fichier ne contient pas d'en-tête :

  5. Dans la zone Preview, cliquez sur le bouton Refresh Preview. Vous avez spécifié le bon délimiteur, le Studio Talend reconnaît que le fichier contient deux colonnes.

    Cliquez surNext :

  6. Dans cette étape, vous spécifiez le schéma du fichier. Modifiez la valeur du champ Name en StateCodesSchema. Modifiez Column0 en StateCode, configurez sa longueur Length à 2 et remplacez Column1 par StateName, puis configurez sa longueur Length à 20.

    Cliquez sur Finish :

  7. Cliquez sur Finish. La nouvelle métadonnée s'affiche dans le Repository :

Étape suivante

Vous pouvez utiliser cette métadonnée afin de créer un composant lisant les codes des États depuis un fichier, pour effectuer une jointure entre ces fichiers et les informations clients.
