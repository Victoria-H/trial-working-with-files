Traitement de fichiers

Vue d'ensemble

Au cours de cette leçon, vous allez commencer à travailler avec des fichiers.

Vous allez d'abord lire un fichier délimité stockant des informations clients depuis votre système de fichiers local. Vous allez ensuite utiliser un composant de transformation de données pour appliquer une modification simple sur les données : passer le nom de l'État du client en majuscules. Pour tester votre transformation, vous allez écrire les données transformées dans un fichier de sortie.

Vous allez finalement consolider vos connaissances sur les transformations de données en combinant les colonnes du prénom et du nom de famille en une colonne contenant le nom complet du client.

Voici la vue haut niveau du Job que vous allez construire dans cette leçon.

 

Objectifs

Après avoir terminé cette leçon, vous pourrez :

  • Lire et écrire un fichier texte délimité
  • Lire des messages d'avertissements et d'erreurs
  • Transformer des données à l'aide du composant tMap
  • Construire des expressions simples
  • Explorer les données à l'aide du Data Viewer dans le Studio
  • Dupliquer un Job existant comme base pour un nouveau Job

Avant de commencer

Assurez-vous de travailler dans un environnement contenant :

  • Une installation fonctionelle du Studio Talend
  • Les fichiers nécessaires à la leçon : Custs.csv dans C:/StudentFiles.

Étape suivante
