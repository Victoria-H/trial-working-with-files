Examen du fichier d'entrée

Avant d'exécuter votre Job, examinez à nouveau le fichier d'entrée.

  1. Utilisez l'explorateur pour trouver le fichier Custs.csv dans le dossier C:/StudentFiles :

  2. Cliquez-droit sur le fichier et sélectionnez Edit with Notepad++ :

    Plusieurs États sont en minuscules. Votre Job est conçu pour les passer en majuscules.
    Exécutez votre Job et vérifiez les résultats.

Exécution

  1. Cliquez sur l'onglet Run sous l'espace de modélisation graphique :

    Vous pouvez exécuter votre Job. Dans l'espace de modélisation graphique, le nom du Job dans son onglet est précédé d'un astérisque, ce qui indique des modifications non sauvegardées.

  2. Cliquez sur le bouton Run :

    Remarquez les statistiques concernant l'exécution du Job, qui s'affichent dans l'espace de modélisation graphique :

    3. Vous pouvez voir que 100 lignes sont passées dans chaque lien.

  3. Via l'explorateur, trouvez et ouvrez le fichier de sortie CappedOut.csv, dans le dossier C:/StudentFiles. Notez que toutes les abréviations d'États sont en majuscules. Fermez le fichier :
