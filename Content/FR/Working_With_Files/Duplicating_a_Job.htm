Vue d'ensemble

Dans les exercices suivants, vous allez agrandir le Job précédent. Plutôt que de commencer à partir de rien, ou de modifier le Job existant et changer sa fonction, vous pouvez le dupliquer et vous en servir comme base pour les exercices suivants.

Duplication

  1. Dans le Repository, sous Job Designs, cliquez-droit surUpperCase et sélectionnez Duplicate :

  2. Saisissez UpperCase_Ex dans le champ Input new name et cliquez sur OK :

    Note : le nouveau Job s'affiche dans le Repository mais ne s'ouvre pas.

Ouvreture

  1. Double-cliquez sur le Job UpperCase_Ex dans le Repository pour l'ouvrir :
