Exercice

Vue d'ensemble

Terminez cet exercice afin de continuer à découvrir le tMap en transformant des données. Consultez la section Solution pour une solution possible à cet exercice.

Combinaison de colonnes

Modifiez le Job UpperCase_Ex précédemment créé afin que la sortie combine les valeurs des numéros et les noms des rues dans une colonne nomméeAddress.

Astuce : vous pouvez consulter la leçon Combinaison de colonnes pour revoir comment vous avez effectué cette action auparavant.

Ajout d'une colonne

Modifiez à nouveau le Job, ajoutez une colonne de type Integer nommée id à la sortie, contenant une valeur d'index automatiquement générée.
Astuce : après avoir ajouté la colonne depuis la fenêtre de configuration du tMap, ouvrez l'Expression Builder et examinez les fonctions de la catégorie Numeric, afin d'en trouver une générant une séquence numérique.

Ajout d'une colonne Zip

Modifiez la configuration du tMap pour que la sortie contienne une nouvelle colonne nommée Zip et contenant un nombre composé de cinq chiffres, aléatoirement généré.
