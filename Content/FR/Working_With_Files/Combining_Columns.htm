Combinaison de colonnes

Vue d'ensemble

Le tMap permet d'effectuer différentes opérations de mapping et transformations de données. Imaginez que votre stockage cible de données nécessite les noms et prénoms combinés en une seule colonne. Suivez les étapes suivantes pour y parvenir.

Modification du schéma

  1. Double-cliquez sur le tMap.

  2. Cliquez dans la case de l'expression row1.Last de la table CappedOut :

    Dans l'éditeur de schéma, au bas de la fenêtre, remarquez que la même colonne est sélectionnée :

    Comme vous allez combiner les données de deux colonnes en une seule, une colonne ne sera plus nécessaire.

  3. Cliquez sur le bouton Remove selected items, marqué d'une croix rouge (encadré dans l'image ci-dessus), pour supprimer la colonne Last du schéma de sortie :

    Important : assurez-vous de cliquer sur le bouton Remove selected items dans l'éditeur du schéma CappedOut (partie inférieure droite de l'éditeur du tMap) et de ne pas utiliser le bouton [x] pour supprimer la table (dans la partie supérieure droite de l'éditeur de mapping). Si vous cliquez sur ce bouton (Remove Table), vous supprimerez votre mapping et devrez tout recommencer.

  4. La colonne a disparu de la table CappedOut en haut de l'éditeur également.

Combinaison de colonnes

  1. Glissez la colonne Last de la table row1 à gauche et déposez-la dans la ligne First de la table CappedOut :

    Notez que l'expression contient à présent des références aux deux colonnes de la tablerow1.

  2. Insérez un signe + entre les deux références :

    Comme les deux colonnes contiennent des chaînes de caractères, le signe + concatène les valeurs.

  3. Pour inclure un espace entre le prénom et le nom de famille, insérez " " + après le +. Appuyez sur la touche Entrée de votre clavier pour sauvegarder cette expression :

  4. Dans l'éditeur de schéma, au bas de la fenêtre, cliquez sur First dans la table CappedOut et changez sa valeur en Name :

    Modifiez la longueur de la ligne Name de 15 à 30.

    Le nom de la colonne du schéma dans la table CappedOut est Name et non First :

  5. Cliquez sur Ok afin de sauvegarder vos modifications et propagez ces changements lorsqu'une boîte de dialogue vous le propose.

Exécution

  1. Pour exécuter le Job, vous pouvez utiliser le bouton Run ou appuyer sur la touche F6 de votre clavier.

  2. Cliquez-droit sur le composant tFileOutputDelimited et sélectionnez Data viewer :

  3. Explorez le contenu du fichier de sortie à l'aide de l'affichage dans la fenêtre Result Data Preview :


    Notez que la première colonne est Name, elle est la combinaison des colonnes First et Last du fichier Custs.csv séparées par un espace (comme spécifié dans l'Expression Builder).

  4. Cliquez dans l'onglet File Content pour voir le format brut du fichier :

    Cliquez sur Close.

Étape suivante

Vous avez terminé ce Job. Vous allez ensuite le dupliquer, le modifier et exécuter le nouveau Job.
