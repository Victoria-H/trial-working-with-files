Solution

Vue d'ensemble

Cette section présente une solution possible à cet exercice. Notez que vos solutions peuvent différer mais être tout aussi valides.

Combinaison de colonnes

Configurez la table CappedOut dans le tMap, supprimez la colonne Number et remplacez la colonne Street par la colonne nommée Address contenant ceci :

row1.Number + " " + row1.Street

Ajout d'une colonne

Ajoutez une colonne de type Int, nommée Id à la table CappedOut, contenant l'expression suivante :

Numeric.sequence("s1",1,1)

Ajout d'une colonne Zip

Cliquez sur la table CappedOut et ajoutez une colonne de type int nommée Zip. Ouvrez l'Expression Builder pour l'expression de la nouvelle colonne, puis créez une expression comme suit :

Numeric.random(10000,99999)

Exécution

Exécutez votre Job et ouvrez le fichierCappedOut.csv pour voir le résultat.
