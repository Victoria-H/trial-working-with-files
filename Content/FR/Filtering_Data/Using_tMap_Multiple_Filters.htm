Utilisation de plusieurs filtres dans le tMap

Vue d'ensemble

Vous allez apprendre à filtrer sur différents États dans un tMap, en suivant les bonnes pratiques. Vous allez créer trois fichiers de sortie, chacun contenant un État différent.

Duplication du Job

  1. Cliquez-droit sur le Job Filter_States et sélectionnez Duplicate :

  2. Nommez le nouveau Job Filter_States_tMap et cliquez sur OK :

  3. Double-cliquez sur le nouveau Job pour l'ouvrir.

Modification de la sortie

  1. Ajoutez un nouveau composant au Job. Cliquez-droit sur le tFileOutputDelimited et sélectionnez Copy :

  2. Cliquez-droit sous le composant de sortie dans l'espace de modélisation graphique et sélectionnez Paste pour placer le composant tFileOutputDelimited :

  3. Le composant est placé sous le tFileOutputDelimited existant.
    Sous le second tFileOutputDelimited, cliquez-droit et sélectionnez Paste pour placer le troisième tFileOutputDelimited à votre Job :

  4. Vous avez placé trois composants de sortie dans votre Job :

  5. Rappelez-vous que le dernier paramètre du tMap était un filtre, filtrant les entrées pour CA (Californie) et écrivant la sortie dans un fichier. Vous allez modifier les paramètres pour le premier composant de sortie et modifier le nom du fichier de sortie. Double-cliquez sur le tFileOutputDelimited_1 et, dans le champ File Name, saisissez "C:/StudentFiles/CA_Out.csv" :

  6. Cliquez sur l'onglet View pour modifier le nom de la sortie en CA. Saisissez CA dans le champ Label format :

  7. Double-cliquez sur le deuxième tFileOutputDelimited, modifiez le nom du fichier en NY_Out.csv et saisissez NY dans le champ Label format :

  8. Effectuez les mêmes modifications pour le troisième composant tFileOutputDelimited et renommez-le AK:

  9. Cliquez-droit sur le composant tMap et sélectionnez Row >*New Output* (Main) :

  10. Reliez-le au composant NY, nommez le lien NYpuis cliquez sur OK :

  11. Cliquez sur Yes lorsqu'une boîte de dialogue vous propose de récupérer le schéma du composant cible :

     

  12. Répétez ces étapes pour le dernier composant de sortie AK :

Modification du tMap

  1. Double-cliquez sur le tMap pour ouvrir son éditeur de mapping.
  2. Cliquez sur le bouton Auto map!, en haut à droite :

  3. Le système va mapper les éléments comme suit :

  4. Vous pouvez copier-coller les expressions. Cliquez-droit sur l'expression dans la ligne Name de la table CappedOut, puis cliquez sur Copy :

  5. Cliquez-droit sur l'expression dans la ligne Name, dans la table NY et sélectionnez Paste :

  6. Collez-la également dans la ligne Name de la table AK :

  7. Copiez-collez l'expression de la ligne State :

  8. Vos tables de sortie doivent ressembler à celles-ci :

  9. Si le filtre n'est pas actif, cliquez sur l'icône représentant une flèche blanche avec un signe + (Activate / unactivate the expression filter), en haut à droite du schéma de sortie CappedOut, puis copiez l'expression "CA".equals(row2.StateCode) :

  10. Cliquez sur l'icône Activate / unactivate the expression filter en haut à droite du schéma de sortie NY et collez l'expression dans le champ qui s'affiche. Modifiez-la en"NY".equals(row2.StateCode) :

  11. Appliquez le mêmes modifications à la sortie AK :

  12. Cliquez sur OK pour sauvegarder vos modifications dans l'éditeur du tMap, puis sauvegardez votre Job.

Exécution du Job

  2. Ouvrez les fichiers de sortie dans le dossier C:/StudentFiles/ afin de vérifier la sortie. Voici le fichier NY_Out.csv :
