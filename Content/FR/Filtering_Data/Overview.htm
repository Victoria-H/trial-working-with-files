Filtre sur les données

Vue d'ensemble

Une tâche commune dans les projets d'intégration de données consiste à filtrer des lignes de données selon leur contenu, pour un traitement séparé, le stockage ou le reporting. Au cours de cette leçon, vous allez construire un Job étendant le Job précédent afin de fusionner les données clients et les données des États puis vous allez séparer les résultats en différents flux de sortie selon la valeur d'une colonne contenant un code pays StateCode :

Objectifs

Après avoir terminé cette leçon, vous pourrez :

  • Utiliser le composant tMap pour filtrer les données

  • Exécuter des parties de Job sous certaines conditions

  • Dupliquer des flux de sortie

Étape suivante

La première étape consiste à ajouter un filtre sur les données afin que seuls les clients d'un État soient écrits dans le fichier de sortie.
